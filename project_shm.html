SHM

A 3D platfrom for strucutural health monitoring

Description

SHM is a platform for structural monitoring, it uses sensors for track stress and displacements; showing information through a 3D interactive user interface. The platform is focused on infrastructure for the mining industry.

It's owned by Mincka, an Australian engineering consultancy firm. I collaborated here with the frontend, developing the 3D UI and REST API integration.

Job Done

  1. Implement 3D UI views based on specified requirements.
  2. Implementation of view cube.
  3. Implementation of rendering structures based in nodes, connections and profiles.
  4. Implementation of zoom in-out, drag scene and selection.
  5. Implementation of dynamic loading 3D scenes.
  6. Implementation of filters based in a range of values.
  7. Integrate REST APIs.
